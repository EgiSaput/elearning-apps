# Panduan Penggunaan Sistem E-Learning Admin

Panduan ini berisi langkah-langkah umum untuk menggunakan sistem E-Learning sebagai admin, mulai dari login, menambah user, mengelola data, hingga meng-assign guru dan siswa.

## 1. Login Admin
- Buka halaman login aplikasi.
- Masukkan username/email dan password admin.
- Klik tombol "Login".

## 2. Menambah User Guru
- Masuk ke dashboard admin.
- Pilih menu "Guru".
- Klik tombol "Tambah Guru".
- Isi data guru: Nama, NIP/NIK, Email, Mata Pelajaran, Foto (opsional), dll.
- Klik "Simpan".

## 3. Menambah User Siswa
- Pilih menu "Siswa" di dashboard admin.
- Klik tombol "Tambah Siswa".
- Isi data siswa: Nama, NISN/NIS, Email, Kelas, Foto (opsional), dll.
- Klik "Simpan".

## 4. Menambah Mata Pelajaran
- Pilih menu "Mata Pelajaran".
- Klik "Tambah Mata Pelajaran".
- Isi nama mapel dan data lain yang diperlukan.
- Klik "Simpan".

## 5. Menambah Kelas
- Pilih menu "Kelas".
- Klik "Tambah Kelas".
- Isi nama kelas dan data lain yang diperlukan.
- Klik "Simpan".

## 6. Assign Guru ke Mata Pelajaran
- Pilih menu "Mata Pelajaran".
- Edit/tambah mapel, pilih guru yang mengajar mapel tersebut.
- Klik "Simpan".

## 7. Assign Siswa ke Kelas
- Pilih menu "Siswa".
- Edit/tambah siswa, pilih kelas siswa.
- Klik "Simpan".

## 8. Mengelola Soal dan Ujian
- Pilih menu "Soal" atau "Ujian".
- Tambah soal/ujian sesuai kebutuhan.
- Atur soal, kunci jawaban, dan pengaturan ujian.
- Klik "Simpan".

## 9. Melihat dan Mengelola Nilai
- Pilih menu "Nilai".
- Lihat nilai siswa, edit jika diperlukan.

## 10. Mengelola Pengumuman
- Pilih menu "Pengumuman".
- Tambah/edit pengumuman untuk siswa/guru.

## 11. Upload File/Gambar
- Pada form yang membutuhkan upload, klik "Pilih File".
- Pilih file dari komputer, lalu klik "Upload".

## 12. Logout
- Klik tombol "Logout" di dashboard admin.

---

**Catatan:**
- Pastikan data yang diinput valid dan tidak duplikat.
- Untuk akses file upload, file disimpan di folder public/upload_gambar.
- Tidak perlu menjalankan symlink storage:link kecuali ada fitur upload yang menggunakan storage Laravel.
- Jika ada error, cek kembali data dan konfigurasi aplikasi.

Panduan ini dapat disesuaikan dengan fitur dan tampilan aplikasi Anda. Jika ingin dibuatkan versi PDF atau lebih detail, silakan informasikan.
