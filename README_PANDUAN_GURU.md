# Panduan Penggunaan Sistem E-Learning Guru

Panduan ini berisi langkah-langkah umum untuk menggunakan sistem E-Learning sebagai Guru.

## 1. Login Guru
- Buka halaman login aplikasi.
- Masukkan email/NIP dan password guru.
- Klik tombol "Login".

## 2. Melihat Jadwal dan Mata Pelajaran
- Setelah login, dashboard guru akan menampilkan jadwal dan daftar mata pelajaran yang diampu.

## 3. Mengelola Soal dan Ujian
- Pilih menu "Soal" atau "Ujian".
- Tambah soal baru, edit soal, atau hapus soal sesuai kebutuhan.
- Atur kunci jawaban dan pengaturan ujian.
- Klik "Simpan".

## 4. Melihat dan Mengelola Nilai Siswa
- Pilih menu "Nilai".
- Lihat nilai siswa untuk mata pelajaran yang diampu.
- Edit atau input nilai jika diperlukan.

## 5. Mengelola Materi dan Tugas
- Pilih menu "Materi" atau "Tugas".
- Upload materi pembelajaran atau tugas untuk siswa.
- Klik "Simpan".

## 6. Melihat Pengumuman
- Pilih menu "Pengumuman" untuk melihat informasi dari admin.

## 7. Mengelola Profil
- Pilih menu "Profil".
- Edit data pribadi dan ubah password jika diperlukan.
- Klik "Simpan".

## 8. Logout
- Klik tombol "Logout" di dashboard guru.

---

**Catatan:**
- Pastikan data yang diinput valid.
- Jika ada error, cek kembali data dan konfigurasi aplikasi.

Panduan ini dapat disesuaikan dengan fitur dan tampilan aplikasi Anda.
