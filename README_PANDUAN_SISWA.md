# Panduan Penggunaan Sistem E-Learning Siswa

Panduan ini berisi langkah-langkah umum untuk menggunakan sistem E-Learning sebagai Siswa.

## 1. Login Siswa
- Buka halaman login aplikasi.
- Masukkan email/NISN/NIS dan password siswa.
- Klik tombol "Login".

## 2. Melihat Jadwal dan Mata Pelajaran
- Setelah login, dashboard siswa akan menampilkan jadwal dan daftar mata pelajaran yang diikuti.

## 3. Mengerjakan Ujian dan Tugas
- Pilih menu "Ujian" atau "Tugas".
- Pilih ujian/tugas yang tersedia.
- Kerjakan soal dan submit jawaban.
- Pastikan jawaban sudah benar sebelum submit.

## 4. Melihat Nilai
- Pilih menu "Nilai".
- Lihat hasil nilai ujian dan tugas yang sudah dikerjakan.

## 5. Melihat Materi
- Pilih menu "Materi" untuk melihat materi pembelajaran yang diupload guru.

## 6. Melihat Pengumuman
- Pilih menu "Pengumuman" untuk melihat informasi dari admin/guru.

## 7. Mengelola Profil
- Pilih menu "Profil".
- Edit data pribadi dan ubah password jika diperlukan.
- Klik "Simpan".

## 8. Logout
- Klik tombol "Logout" di dashboard siswa.

---

**Catatan:**
- Pastikan data yang diinput valid.
- Jika ada error, cek kembali data dan konfigurasi aplikasi.

Panduan ini dapat disesuaikan dengan fitur dan tampilan aplikasi Anda.
